# Firebase Functions Setup for Automation

This guide explains how to set up Firebase Cloud Functions for scheduled automations (monthly invoices, daily reminders, etc.).

## Prerequisites

1. Install Firebase CLI:
```bash
npm install -g firebase-tools
```

2. Login to Firebase:
```bash
firebase login
```

3. Initialize Firebase Functions in your project:
```bash
firebase init functions
```

Select:
- TypeScript
- ESLint (yes)
- Install dependencies (yes)

After initialization, you'll have:

```
marketflow-dashboard-main/
```

```typescript
import * as functions from 'firebase-functions';
import * as admin from 'firebase-admin';

admin.initializeApp();

const db = admin.firestore();

// Monthly Invoice Generation
export const generateMonthlyInvoices = functions.pubsub
  .schedule('0 0 1 * *') // 1st of every month at midnight
  .timeZone('America/New_York')
  .onRun(async (context) => {
    const now = new Date();
    const invoiceMonth = now.toLocaleDateString('en-US', { month: 'long', year: 'numeric' });
    const dueDate = new Date(now.getFullYear(), now.getMonth(), 15).toLocaleDateString();

    const budgetAmounts: Record<string, number> = {
      starter: 750,
      growth: 2000,
      scale: 4000,
      enterprise: 6000,
    };

    // Get all active clients
    const clientsSnapshot = await db.collection('clients')
      .where('projectStatus', '==', 'Active')
      .get();

    // Get existing invoices for this month
    const invoicesSnapshot = await db.collection('invoices')
      .where('month', '==', invoiceMonth)
      .get();

    const existingClientIds = new Set(
      invoicesSnapshot.docs
        .map(doc => doc.data().clientId || doc.data().clientEmail)
        .filter(Boolean)
    );

    let created = 0;
    let skipped = 0;

    for (const clientDoc of clientsSnapshot.docs) {
      const client = clientDoc.data();
      const clientId = clientDoc.id;

      // Skip if invoice already exists
      if (existingClientIds.has(clientId) || (client.email && existingClientIds.has(client.email))) {
        skipped++;
        continue;
      }

      // Calculate invoice amount
      const amount = client.budget ? (budgetAmounts[client.budget] || 1000) : 1000;

      // Create invoice
      await db.collection('invoices').add({
        clientName: client.businessName || client.email || 'Client',
        clientEmail: client.email,
        clientId: clientId,
        amount: amount,
        status: 'pending',
        month: invoiceMonth,
        dueDate: dueDate,
        createdAt: admin.firestore.FieldValue.serverTimestamp(),
        autoGenerated: true,
      });

      // Log activity
      await db.collection('activities').add({
        type: 'invoice_created',
        title: `Invoice auto-generated for ${client.businessName || client.email}`,
        subtitle: `Amount: $${amount}, Month: ${invoiceMonth}`,
        clientId: clientId,
        metadata: { amount, month: invoiceMonth, autoGenerated: true },
        ts: Date.now(),
        createdAt: admin.firestore.FieldValue.serverTimestamp(),
      });

      created++;
    }

    console.log(`Generated ${created} invoices, skipped ${skipped}`);
    return { created, skipped };
  });

// Daily Payment Reminders
export const checkPaymentReminders = functions.pubsub
  .schedule('0 9 * * *') // 9 AM daily
  .timeZone('America/New_York')
  .onRun(async (context) => {
    const now = new Date();
    const today = now.toISOString().split('T')[0];

    // Get all pending invoices
    const invoicesSnapshot = await db.collection('invoices')
      .where('status', '==', 'pending')
      .get();

    let remindersSent = 0;
    let overdueUpdated = 0;

    for (const invoiceDoc of invoicesSnapshot.docs) {
      const invoice = invoiceDoc.data();
      if (!invoice.dueDate) continue;

      const dueDate = new Date(invoice.dueDate);
      const daysUntilDue = Math.ceil((dueDate.getTime() - now.getTime()) / (1000 * 60 * 60 * 24));
      const daysOverdue = -daysUntilDue;

      // Update to overdue if past due date
      if (daysOverdue > 0 && invoice.status === 'pending') {
        await invoiceDoc.ref.update({ status: 'overdue' });

        await db.collection('activities').add({
          type: 'invoice_created',
          title: `Invoice marked as overdue: ${invoice.clientName}`,
          subtitle: `Amount: $${invoice.amount}, ${daysOverdue} day${daysOverdue > 1 ? 's' : ''} overdue`,
          clientId: invoice.clientId || '',
          metadata: { invoiceId: invoiceDoc.id, daysOverdue },
          ts: Date.now(),
          createdAt: admin.firestore.FieldValue.serverTimestamp(),
        });

        overdueUpdated++;
      }

      // Send reminders based on timing
      if (daysUntilDue === 3 || daysUntilDue === 0 || daysOverdue === 3 || daysOverdue === 7) {
        await db.collection('activities').add({
          type: 'invoice_created',
          title: `Payment reminder: ${invoice.clientName}`,
          subtitle: `Invoice ${invoice.month} - $${invoice.amount} ${daysOverdue > 0 ? `(${daysOverdue} days overdue)` : `(due in ${daysUntilDue} days)`}`,
          clientId: invoice.clientId || '',
          metadata: {
            invoiceId: invoiceDoc.id,
            reminderType: daysUntilDue === 3 ? 'before' : daysOverdue > 0 ? 'overdue' : 'due',
          },
          ts: Date.now(),
          createdAt: admin.firestore.FieldValue.serverTimestamp(),
        });

        remindersSent++;
      }
    }

    console.log(`Sent ${remindersSent} reminders, updated ${overdueUpdated} overdue invoices`);
    return { remindersSent, overdueUpdated };
  });

// Firestore Trigger: Auto-generate tasks from weekly plan
export const onWeeklyPlanUpdate = functions.firestore
  .document('weeklyPlans/{planId}')
  .onWrite(async (change, context) => {
    const planData = change.after.exists ? change.after.data() : null;
    if (!planData) return null;

    const clientId = planData.clientId;
    if (!clientId || !planData.weeks || planData.weeks.length === 0) return null;

    // Get existing tasks
    const existingTasksSnapshot = await db.collection('tasks')
      .where('clientId', '==', clientId)
      .get();

    const existingTasks = existingTasksSnapshot.docs.map(doc => ({
      id: doc.id,
      ...doc.data(),
    }));

    let created = 0;

    // Generate tasks
    for (const week of planData.weeks) {
      for (const taskTitle of week.tasks) {
        if (!taskTitle.trim()) continue;

        // Check if task already exists
        const exists = existingTasks.some(
          (t: any) =>
            t.clientId === clientId &&
            t.title === taskTitle.trim() &&
            t.week === week.weekLabel
        );

        if (!exists) {
          await db.collection('tasks').add({
            title: taskTitle.trim(),
            priority: 'medium',
            status: 'todo',
            clientId: clientId,
            week: week.weekLabel,
            createdAt: admin.firestore.FieldValue.serverTimestamp(),
            createdBy: 'automation',
            autoGenerated: true,
          });
          created++;
        }
      }
    }

    if (created > 0) {
      await db.collection('activities').add({
        type: 'task_created',
        title: `${created} task${created > 1 ? 's' : ''} auto-generated from weekly plan`,
        subtitle: `Client: ${clientId}`,
        clientId: clientId,
        metadata: { taskCount: created, fromWeeklyPlan: true, autoGenerated: true },
        ts: Date.now(),
        createdAt: admin.firestore.FieldValue.serverTimestamp(),
      });
    }

    return { created };
  });

// Firestore Trigger: Auto-assign onboarding tasks
export const onClientOnboardingComplete = functions.firestore
  .document('clients/{clientId}')
  .onUpdate(async (change, context) => {
    const before = change.before.data();
    const after = change.after.data();
    const clientId = context.params.clientId;

    // Check if onboarding was just completed
    if (!before.onboardingCompleted && after.onboardingCompleted) {
      // Check if tasks already exist
      const existingTasksSnapshot = await db.collection('tasks')
        .where('clientId', '==', clientId)
        .get();

      if (!existingTasksSnapshot.empty) {
        return { created: 0 }; // Tasks already exist
      }

      // Default onboarding tasks
      const onboardingTasks = [
        'Review marketing strategy',
        'Complete business profile',
        'Set up initial goals',
        'Schedule kickoff meeting',
      ];

      let created = 0;

      for (const taskTitle of onboardingTasks) {
        await db.collection('tasks').add({
          title: taskTitle,
          priority: 'high',
          status: 'todo',
          clientId: clientId,
          createdAt: admin.firestore.FieldValue.serverTimestamp(),
          createdBy: 'automation',
          autoGenerated: true,
          isOnboardingTask: true,
        });
        created++;
      }

      if (created > 0) {
        await db.collection('activities').add({
          type: 'task_created',
          title: `${created} onboarding task${created > 1 ? 's' : ''} auto-assigned`,
          subtitle: `Client: ${after.businessName || after.email || clientId}`,
          clientId: clientId,
          metadata: { taskCount: created, autoGenerated: true, isOnboardingTask: true },
          ts: Date.now(),
          createdAt: admin.firestore.FieldValue.serverTimestamp(),
        });
      }

      return { created };
    }

    return null;
  });
```

## Deployment

1. Build the functions:
```bash
cd functions
npm run build
```

2. Deploy all functions:
```bash
firebase deploy --only functions
```

3. Deploy specific function:
```bash
firebase deploy --only functions:generateMonthlyInvoices
```

## Testing Locally

1. Start the emulator:
```bash
firebase emulators:start --only functions
```

2. Test scheduled functions manually:
```bash

```


## Cost Considerations

- **Free Tier**: 2 million invocations/month, 400,000 GB-seconds, 200,000 CPU-seconds
- **Pricing**: $0.40 per million invocations after free tier
- Scheduled functions run once per schedule (monthly = 12 invocations/year)
- Firestore triggers run on every document change

## Notes

- Functions run in UTC by default, adjust timezone in schedule
- Firestore triggers have a 60-second timeout
- Scheduled functions have a 540-second timeout
- Use batch writes for multiple document operations
- Consider rate limits when processing large datasets

## Troubleshooting

1. **Function not triggering**: Check Firebase Console > Functions > Logs
2. **Permission errors**: Ensure service account has Firestore permissions
3. **Timeout errors**: Optimize queries, use batch operations
4. **Billing issues**: Check Firebase Console > Usage and Billing
